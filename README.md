# RCM MDM(Master Data Management)

## Development

Install dependencies and build the components:

```bash
npm install
```

Configure `.env` file in every service package:

```
PORT=3000
REACT_APP_BASEAPI_URL=
REACT_APP_MARKETPLACE_REDIRECT=
REACT_APP_IDENTITYAPI_URL=""
```

Start React Application:

```bash
npm start
```

Build React Application:

```bash
npm run build
```
